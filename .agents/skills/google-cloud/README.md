# Google Cloud Skills

This category mirrors the official Google skill pack from `google/skills`.

Included skills:

- `gemini-api`
- `gemini-agents-api`
- `agent-platform-skill-registry`

Why it belongs:

- `gemini-api` covers Gemini usage on Google Cloud Agent Platform, which is the project's model and reasoning surface.
- `gemini-agents-api` covers managed agent resources and the control plane used to define stateful agent behavior.
- `agent-platform-skill-registry` covers skill discovery and skill lifecycle operations for agent-oriented workflows.
