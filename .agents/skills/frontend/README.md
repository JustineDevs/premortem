# Frontend Skills

This category mirrors official Vercel skill packs for the React and Next.js surface used by the Premortem control plane.

Included skills:

- `next-best-practices`
- `react-best-practices`

Why it belongs:

- `next-best-practices` covers the Next.js conventions, runtime boundaries, data patterns, and routing behavior used by the app.
- `react-best-practices` covers React performance and rendering patterns that matter for the operator-facing UI.
