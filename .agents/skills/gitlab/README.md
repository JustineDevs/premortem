# GitLab Skills

This category mirrors official GitLab skill packs from `gitlab-org/ai/skills` and `gitlab-org/orbit/knowledge-graph`.

Included skills:

- `glab`
- `glab-glql`
- `orbit`

Why it belongs:

- `glab` covers the GitLab CLI workflow used for issues, merge requests, and API-backed automation.
- `glab-glql` covers GitLab GraphQL-style query workflows where the project needs structured reads.
- `orbit` covers the GitLab Knowledge Graph, which is the best-fit official surface for repo graph and dependency analysis.
