# Code of Conduct

## Our standards

We want this project to be respectful, constructive, and safe for contributors and users.

Examples of expected behavior:
- Be respectful and professional.
- Give clear, actionable feedback.
- Assume good intent while still being precise.
- Focus criticism on code, design, and decisions, not people.
- Help keep discussions accessible and welcoming.

Examples of unacceptable behavior:
- Harassment, insults, or personal attacks.
- Discriminatory or exclusionary language.
- Publishing private information without permission.
- Bad-faith disruption, intimidation, or hostile conduct.

## Scope

This code of conduct applies to project spaces, discussions, issues, pull requests, reviews, and related community interactions.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that violate this code of conduct.

## Reporting

If someone is violating the code of conduct, report it privately to the maintainers with enough context for review.
